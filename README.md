# 💰 Daily Expense Tracker (Java OOP Project)

## 📌 Overview
The Daily Expense Tracker is a Java-based desktop application developed using Object-Oriented Programming (OOP) concepts.  
It helps users manage their daily expenses efficiently by allowing them to record, view, and analyze spending.

This project focuses on simplicity, usability, and structured data management using file handling.

---

## 🎯 Objectives
- Track daily expenses easily  
- Maintain organized financial records  
- Provide a simple and user-friendly interface  
- Demonstrate OOP concepts in a real-world application  

---

## ✨ Features

### 🔐 User Authentication
- Signup with email, user ID, and password  
- Login validation using stored data  

### 💵 Expense Management
- Add expenses with:
  - Date  
  - Category  
  - Amount  
  - Description  
- Delete selected expenses  
- View all expenses in tabular format  

### 📊 Data Handling
- File-based storage system  
- Persistent data across sessions  

### 📈 Additional Functionalities
- Automatic total expense calculation  
- Structured UI using Java Swing  

---

## 🧠 OOP Concepts Used
- **Encapsulation** → User & Expense classes  
- **Abstraction** → FileManager & ExpenseManager  
- **Modularity** → Separation into packages (model, service, ui)
